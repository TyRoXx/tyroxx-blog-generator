# Installation
* Get conan with "pip install conan"
* install the package: cmake, build-essential
* run 'conan install [source] --build silicium --build ventura'
* run 'cmake -G "CodeBlocks - Unix Makefiles" [source]'
* requires gcc version >= 4.9
